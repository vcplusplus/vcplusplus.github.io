Resources

These are some useful links and resources. Most of them are for our members but you might find something useful too.

  • Our Github: https://github.com/vcplusplus

    Throughout the year our members will post tutorials with sample code on the VC++ Github page. You can fork the repository and add to it to make it your own. This is a great way to learn how to read code as well as improve your coding skills.
